#Expense Tracker

##Project Description

This Expense Tracker was developed as Task 2 assigned during my Python Programming Internship. The objective was to create a program that accepts multiple expense inputs from the user, calculates the total expenditure, and displays the final amount.

##Features

- Accepts multiple expense entries
- Calculates the total amount spent
- Uses basic Python concepts such as loops, variables, and arithmetic operations
- Simple and easy-to-understand command-line interface

##Technologies Used

- Python 3

##How to Run

1. Download the project files.
2. Open the project in Visual Studio Code or any Python IDE.
3. Run the Python script.
4. Enter expense values when prompted.
5. View the calculated total expense.

##Outcome

This task helped strengthen my understanding of Python programming fundamentals, user input handling, and mathematical operations through a practical application.
